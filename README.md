# AI-Enhanced ICU Patient Monitoring & Clinical Decision Support Dashboard

## Overview
This project is a MATLAB App Designer-based intelligent ICU patient monitoring dashboard that simulates real-time physiological monitoring and clinical decision support.

## Features
- Real-time ECG waveform simulation
- Heart Rate monitoring
- SpO₂ monitoring
- Blood Pressure monitoring
- Temperature monitoring
- Respiratory Rate monitoring
- Patient Severity Score calculation
- LOW / MEDIUM / CRITICAL risk classification
- Clinical recommendation engine
- Sensor anomaly detection
- Emergency CODE BLUE alert system
- Patient trend forecasting
- Live system clock
- Start / Pause / Resume monitoring controls

## Technologies Used
- MATLAB
- MATLAB App Designer

## Project Highlights
This dashboard simulates ICU patient monitoring with intelligent decision-support logic, emergency alert handling, and biomedical signal visualization.
